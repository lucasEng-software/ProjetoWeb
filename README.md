##Status da API -
* 1: Status 200 (OK)
* 2: Status 201 (Created)
* 3: Status 409 (Conflict) - quando houver um usuário ja cadastrado
* 4: Status 404 (Not Found) 
* 5: Status 400 (Bad Request)
